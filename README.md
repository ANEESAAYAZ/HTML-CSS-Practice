# HTML & CSS Practice Exercises

This repository contains my HTML and CSS practice exercises created while learning front-end web development. The projects focus on building a strong foundation in HTML structure, CSS styling, and responsive web design.

## 📚 Topics Covered

- HTML5 Basics
- HTML Tags
- Headings and Paragraphs
- Images
- Hyperlinks
- Lists (Ordered & Unordered)
- Tables
- Forms
- Colors and Backgrounds
- Fonts and Text Styling
- Box Model
- Margins and Padding
- Borders
- CSS Classes and IDs
- Layout and Positioning

## 🛠 Technologies Used

- HTML5
- CSS3
- Visual Studio Code

## 📂 Repository Structure

Each folder contains a separate exercise or mini project demonstrating different HTML and CSS concepts.

## 🎯 Purpose

The goal of this repository is to improve my front-end development skills by practicing HTML and CSS through hands-on exercises and small projects.
